SNDtools
====
## Description
CLI sound processing tools written in C++.

## Requirements
- [boost](http://www.boost.org/)
- [portaudio](http://www.portaudio.com/)
- [libsndfile](http://www.mega-nerd.com/libsndfile/)
- [fftw](http://www.fftw.org/)
